## Function
* Create a function:
{% highlight javascript %}
var timesTwo = function(number) {
    return number * 2;
};
{% endhighlight %}

Syntax:
{% highlight javascript %}
var functionName = function(input) {
    /do something
};
{% endhighlight %}
